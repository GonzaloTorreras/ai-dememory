# Public release provenance

This repository starts from a reviewed source snapshot rather than the private
development repository's Git history.

- Private source commit: `cf5a4f55b9c91993e4077c2103e6a4ea2ee4bbaf`
- Private source tree: `d47ef4d04791c8724fddf24c989db566f43cc37b`
- Export policy: explicit source and documentation allowlist
- Excluded: private Git history, generated inbox captures, local reports,
  indexes, working state, credentials and non-demo memory

The files under `memories/` are reviewed demonstration fixtures. Users should
keep their real vault in a separate private repository or unversioned path.

The initial public commit should remain the only root commit. Future releases
follow the checks and approval gates documented in this repository.
