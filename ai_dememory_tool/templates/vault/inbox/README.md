# Inbox

Use this directory for unreviewed captures.

LLM-written proposals should go under `llm-captures/`.

Other review inboxes:

- `imports/`: provider, text, and Markdown import candidates.
- `recall-feedback/`: reviewed search miss candidates.
- `release-acceptance/`: reviewed manual release acceptance evidence.
- `session-events/`: hook event metadata.
- `sleep-consolidation/`: reviewed sleep consolidation packets.
- `conflict-resolution/`: reviewed conflict merge proposals.
- `review-recommendations/`: advisory LLM/client review recommendations.
